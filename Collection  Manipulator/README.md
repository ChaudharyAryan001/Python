# Student Data Organizer

## Features
- Add new student records with ID, name, age, DOB, grade, and subjects
- Display all stored student information
- Update student details (age and subjects)
- Delete student records by ID
- Display unique subjects offered across all students
- Exit option for program termination

## Program Flow
1. Display welcome message and main menu
2. User selects an option:
   - **Option 1:** Add a new student
   - **Option 2:** Display all students
   - **Option 3:** Update student information
   - **Option 4:** Delete a student record
   - **Option 5:** Display unique subjects offered
   - **Option 6:** Exit the program
3. Execute the selected functionality
4. Show results or confirmation messages
5. Continue until user chooses to exit

## Concepts Used
- Lists for storing student records
- Dictionaries for organizing student attributes
- Tuples for storing immutable ID and DOB information
- Loops (`while`, `for`) for menu and data traversal
- Conditional statements (`if`, `elif`, `else`) for option handling
- String operations (splitting, joining, stripping)
- Set operations for unique subject collection
- User input handling (`input()`)
