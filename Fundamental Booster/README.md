Personal Data Collector
Features
Collects user input: name, age, height, and favourite number
Calculates birth year from age
Displays each value with its data type and memory address
Prints a summary of the collected information
Program Flow
Display program title
Take user input
Calculate birth year using age
Display all information with type and memory address
End program with a message
Concepts Used
Input and Output
Data Types (int, float, str)
Type checking using type()
Memory address using id()
Arithmetic operations
Formatted strings (f-strings)
